컬렉션 프레임워크
    - 컴퓨터의 자료구조를 자바로 표현한 클래스/인터페이스
    - 각 상황에 따라 효율적인 자료구조를 선택해서 프로그램을 만들기
[과제]
    1. ArrayList
    2. Vector
    3. LinkedList
    4. HashSet
    5. HashMap
    6. Stack
    7. Queue
    8. TreeSet
    9. TreeMap

    - 9가지의 클래스/인터페이스들을 사용하는 상황을 1개 이상씩 작성
        e.g. ~~ : ~~을 사용하므로 ~~할 때 효율적이다
    - 9가지의 클래스/인터페이스들의 메모리 구조를 시각화해서 표현 (PPT/CANVA)
