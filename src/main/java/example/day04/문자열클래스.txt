        char[] 배열변수 = new char[3];
        char[0] = '유'; char[1] = '재'; char[2] = '석';
[2] 자바 문자열 : String 클래스 지원
    1. String 변수명 = new String("abc");
    2. String 변수명 = "abc";
[3] 문자열 연결하는 법
    1. 문자열1.concat(문자열2)
    2. 문자열1 += 문자열2
    3. StringBuilder 변수명 = new StringBuilder();
        변수명.append(문자열);
    차이 : 새로운 문자열 반환 여부
        - StringBuilder 클래스는 기존 메모리를 그대로 연결해서 사용
[4] String 문자열에서 자주 사용하는 함수들
    1.문자열.charAt(index) : 해당 인덱스의 문자 1개 반환
    2.문자열.length() : 해당 문자열의 길이
    3.문자열.replace(기존문자열, 변경할문자열) : 기존 문자열을 변경해서 새로운 문자열로 반환
    4.문자열.subString(시작인덱스, [끝인덱스]) : 끝 인덱스 바로 전까지 잘라 새로운 문자열로 반환
    5.문자열.split("구분 문자") : 구분 문자 기준으로 문자열 분해해서 배열로 반환
    6.문자열.indexOf("찾을문자열") : 찾을 문자가 있으면 인덱스 반환, 없으면 -1 반환
    7.문자열.contains("찾을문자열") : 찾을 문자 여부에 따라 true/false
    8.문자열.getBytes() : 문자열 내 문자 하나씩 바이트로 변형된 바이트 배열로 변환

[5] 문자와 바이트 관계, 자바 : 유니코드 UTF-8 사용 (한글도 표현가능)
    - 모든 문자는 아스키코드 규약으로 바이트를 문자로 표현
    - 97 --> 'a', 44032 --> '가'
    - char타입 : ''작은 따옴표로 감싼 타입, 문자 1개, 65536개 글자 표현, 2바이트, unsigned
